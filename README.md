# Till We Meet Again - Premium Caskets Website

## Project Overview
A modern, respectful website for a premium casket business using Next.js, Tailwind CSS, and shadcn/ui components.

## Current Implementation

### Pages
1. **Home Page (`/`):**
   - Responsive header with contact information
   - Navigation menu
   - Hero section with business tagline
   - Product grid showcasing 8 caskets
   - Footer with copyright information

2. **About Page (`/about`):**
   - Mission statement
   - Company story
   - Team section with 3 placeholder members

3. **Products Page (`/products`):**
   - Coming soon message
   - Placeholder for future product catalog

### Components
- `NavLink`: Reusable navigation link component
- `ProductCard`: Card component for displaying casket products
- `TeamMember`: Card component for team member profiles
- `StyledLayout`: Global styling wrapper component

### Styling
- Dark theme with green accents
- Glassmorphism UI effects
- Responsive design for all screen sizes
- Custom fonts: Playfair Display for headings, Inter for body text

- Next.js 13.5.1
